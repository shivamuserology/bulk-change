# Bulk Change Prototype

Interactive prototype demonstrating a bulk employee attribute change capability.

## Live Demo

https://shivamuserology.github.io/bulk-change

## Features

- 7-step wizard flow for bulk changes
- Multiple entry modes (UI-guided, CSV upload)
- Permission handling (full access, approval required, restricted)
- Outcome scenarios (happy path, warnings, errors, partial failure)
- Demo control panel for switching scenarios

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run deploy
```
